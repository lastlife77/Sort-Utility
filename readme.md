# Sort Utility

Упрощённая клон-утилита командной строки sort из UNIX-систем, написанная на Go. Сортирует строки текста из файла или стандартного ввода с поддержкой различных алгоритмов сравнения.

# Возможности

- Сортировка по всей строке или конкретной колонке.
- Поддержка числовой, строковой и сортировки по месяцам.
- Обратный порядок сортировки и вывод уникальных строк.
- Проверка отсортированности данных.
- Поддержка "человекочитаемых" числовых суффиксов (K, M, G).
- Чтение из файла или стандартного ввода (stdin).

# Флаги

- **-k N** — сортировать по столбцу (колонке) №N (разделитель — табуляция по умолчанию).
  Например, «sort -k 2» отсортирует строки по второму столбцу каждой строки.
- **-n** — сортировать по числовому значению (строки интерпретируются как числа).
- **-r** — сортировать в обратном порядке (reverse).
- **-u** — не выводить повторяющиеся строки (только уникальные).
- **-M** — сортировать по названию месяца (Jan, Feb, ... Dec), т.е. распознавать специфический формат дат.
- **-b** — игнорировать хвостовые пробелы (trailing blanks).
- **-c** — проверить, отсортированы ли данные; если нет, вывести сообщение об этом.
- **-h** — сортировать по числовому значению с учётом суффиксов (например, К = килобайт, М = мегабайт — человекочитаемые размеры).

## Примеры использования

Сортировка файла по всей строке:

```bash
sort input.txt
```

Сортировка по колонке (табуляция — разделитель по умолчанию):

```bash
sort -k=2 input.txt
```

Числовая сортировка:

```bash
sort -n numbers.txt
```

Обратный порядок и уникальные строки:

```bash
sort -ru input.txt
```

Сортировка по месяцам:

```bash
sort -M months.txt
```

Сортировка чисел с суффиксами K, M, G:

```bash
sort -H sizes.txt
```

Проверка, отсортированы ли данные:

```bash
sort -c input.txt
```

Чтение из стандартного ввода:

```bash
sort -nrk=2
```
